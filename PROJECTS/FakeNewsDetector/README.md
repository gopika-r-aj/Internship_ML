#  Fake News Detector using Machine Learning

This project is a **Fake News Detection System** that uses **Natural Language Processing (NLP)** and **Machine Learning** to classify news articles as **Fake** or **Real**. It includes a Streamlit web application that allows users to input news text and get instant predictions.

##  Features

- Preprocessing of news data (stopword removal, lemmatization, etc.)
- TF-IDF vectorization
- ML model (e.g., Logistic Regression, Random Forest, etc.)
- Real-time prediction using Streamlit UI
- Clean and user-friendly interface


##  Technologies Used

- Python 3.11
- Scikit-learn
- Pandas, Numpy
- Streamlit
- NLTK / spaCy
- Git & GitHub
